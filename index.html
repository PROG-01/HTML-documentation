Introduction

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It consists of elements that tell browsers how to display text, images, links, and other types of content. This documentation provides a reference for many commonly used HTML elements. Each section includes a description of the element, its purpose, and example code.

Text Elements

HTML provides a variety of text elements that define the structure and emphasis of content. These elements help browsers, screen readers, and search engines understand the meaning and importance of different parts of the text.

Headings

Headings are defined using tags from <h1> to <h6>, where <h1> is the highest (most important) level and <h6> is the lowest. They should be used in a logical, hierarchical order.

                                
                                    <h1>Main Title</h1>
                                    <h2>Section Heading</h2>
                                    <h3>Subsection Heading</h3>

Paragraphs

The <p> tag is used to define blocks of text as paragraphs. It adds spacing above and below the content by default.

                                
                                    <p>This is a paragraph of text.</p>

Line Breaks

Use the <br> tag to insert a line break without starting a new paragraph. It’s an empty (self-closing) tag.

                                This is the first line.<br>This is the second line.

Text Emphasis

HTML provides elements to emphasize or highlight the importance of certain text:

  • <strong> – indicates strong importance (usually bold)
  • <em> – indicates emphasis (usually italic)
                                
                                    <p>This is <strong>very important</strong>.</p>
                                    <p>This word is <em>emphasized</em>.</p>

Lists

HTML provides elements for creating both unordered (bulleted) and ordered (numbered) lists. Lists help organize related items and improve content readability.

Unordered Lists

An unordered list is created using the <ul> element. Each list item is defined with <li>. By default, browsers display bullets for unordered list items

                                
                                    <ul>
                                    <li>Apples</li>
                                    <li>Bananas</li>
                                    <li>Cherries</li>
                                    </ul>

Ordered Lists

An ordered list is created using the <ol> element, with each list item again using <li>. Items are numbered automatically.

                                
                                    <ol>
                                    <li>Wake up</li>
                                    <li>Brush teeth</li>
                                    <li>Start coding</li>
                                    </ol>

Nested Lists

Lists can be nested inside each other to create multi-level structures. This is useful for representing subcategories or hierarchies.

Links and Images

HTML allows you to link to other web pages or resources using the anchor tag, and to embed images using the image tag. These elements are essential for building rich, interactive websites.

Anchor Tag (Links)

The <a> (anchor) element creates a hyperlink. The href attribute specifies the URL the link goes to. You can also use the target attribute to open links in a new tab.

                          
                            <a href="https://www.example.com">Visit Example</a>

Open in a new tab:

                              
                                <a href="https://www.example.com" target="_blank">Visit Example</a>

Image Tag

The <img> tag embeds images into the page. It requires a src attribute (the image URL or path) and should always include an alt attribute for accessibility and SEO.

                              
                                <img src="logo.png" alt="Company logo">

Clickable Images

You can wrap an image in an anchor tag to make it clickable, turning it into a link.

                              
                                <a href="https://example.com">
                                <img src="logo.png" alt="Example logo">
                                </a>

Forms

Forms in HTML are used to gather input from users. They consist of form elements like text fields, checkboxes, radio buttons, dropdowns, and buttons. Data entered into a form can be sent to a server for processing.

Basic Form Structure

The <form> element wraps all form controls. Use the action attribute to specify where the form data should go, and method (usually GET or POST) to define how it’s sent.

                          
                            <form action="/submit" method="post">
                            <!-- Form inputs go here -->
                            </form>

Input Fields

Use the <input> element to create different types of user inputs such as text fields, emails, and passwords.

                          
                            <label>
                            Name: <input type="text" name="username">
                            </label>

                            <label>
                            Email: <input type="email" name="email">
                            </label>

Textarea

The <textarea> tag is used for multi-line text input.

                          
                            <label>
                            Message:
                            <textarea name="message"></textarea>
                            </label>

Radio Buttons and Checkboxes

Use type="radio" for single-choice options and type="checkbox" for multiple-choice

                          
                            <p>Choose a language:</p>
                            <label><input type="radio" name="language" value="html"> HTML</label>
                            <label><input type="radio" name="language" value="css"> CSS</label>

                            <p>Select your skills:</p>
                            <label><input type="checkbox" name="skill" value="javascript"> JavaScript</label>
                            <label><input type="checkbox" name="skill" value="python"> Python</label>

Submit Button

The <button> or <input type="submit"> triggers form submission.

                          
                            <button type="submit">Send</button>

Tables

Tables are used to display tabular data — information in rows and columns.

Basic Table Structure

Here's an example of a simple table:

                            
                              <table>
                              <thead>
                              <tr>
                              <th>Element</th>
                              <th>Description</th>
                              </tr>
                              </thead>
                              <tbody>
                              <tr>
                              <td><table></td>
                              <td>Defines a table</td>
                              </tr>
                              <tr>
                              <td><tr></td>
                              <td>Defines a row</td>
                              </tr>
                              <tr>
                              <td><td></td>
                              <td>Defines a cell</td>
                              </tr>
                              </tbody>
                              </table>

Rendered result:

Element Description
<table> Defines a table
<tr> Defines a row
<td> Defines a table cell

Semantic Tags

Semantic HTML tags clearly describe their meaning in a way that is understandable to both developers and browsers. They improve accessibility and search engine optimization (SEO).

Common Semantic Elements

  • <header> – Represents introductory content, often including a logo or navigation.
  • <nav> – Defines a block of navigation links.
  • <main> – The central content of a page.
  • <section> – A generic section of related content, often with a heading.
  • <article> – A self-contained piece of content, like a blog post or news item.
  • <aside> – Sidebar content related to the main content.
  • <footer> – Footer content for a section or page.

Example Usage

                          
                            <article>
                            <header>
                            <h2>Semantic HTML</h2>
                            </header>
                            <p>This is a paragraph inside an article element.</p>
                            <footer>Posted by Admin</footer>
                            </article>

Multimedia Elements

HTML supports multimedia content such as audio, video, and images. These elements enrich web pages and improve user experience.

Image Element

The <img> tag is used to embed images. Always use the alt attribute for accessibility.

                          
                            <img src="cat.jpg" alt="A cute cat">

Audio Element

The <audio> tag lets you embed sound content. Use the controls attribute to show audio controls.

                          <audio controls>
                            <source src="sound.mp3" type="audio/mpeg">
                            Your browser does not support the audio element.
                            </audio>

Video Element

The <video> tag is used to play video files. You can specify width, height, and add controls.

                          <video width="320" height="240" controls>
                           <source src="video.mp4" type="video/mp4">
                           Your browser does not support the video tag.
                           </video>

HTML Document Structure

A basic HTML document follows a standard structure that ensures it is interpreted correctly by browsers and other user agents.

<!DOCTYPE html>
             
Basic Structure

                 <html lang="en">
                 <head>
                 <meta charset="UTF-8">
                 <meta name="viewport" content="width=device-width, initial-scale=1.0">
                 <title>Page Title</title>
                 </head>
                 <body>
                 <!-- Page content goes here -->
                 </body>
                 </html>

              
Key Parts Explained

                
    
                  
  • <!DOCTYPE html> – Declares the document type and version of HTML.
    • 
                  
  • <html> – Root element of the document.
    • 
                  
  • <head> – Contains meta-information, links, and the title of the page.
    • 
                  
  • <meta> – Provides metadata like character encoding and responsive settings.
    • 
                  
  • <title> – Sets the title of the web page shown in the browser tab.
    • 
                  
  • <body> – Holds the content that is displayed in the browser.
